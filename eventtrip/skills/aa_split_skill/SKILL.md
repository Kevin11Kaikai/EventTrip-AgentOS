# AA Split Skill

Use this skill to keep collaborative trip costs fair and explicit.

## Cost Ownership

- Individual flights are paid separately by each traveler.
- Individual tickets are paid separately by each traveler.
- Shared hotel costs are split equally.
- Shared Uber, taxi, or local transportation costs are split equally.
- Optional individual upgrades are not shared unless agreed in advance.

## Cancellation Responsibility

1. Confirm cancellation rules before booking shared costs.
2. Identify the booking owner for each shared reservation.
3. Decide whether a canceling traveler reimburses non-refundable shared costs.
4. Put the AA agreement in writing before money is spent.

## Booking Owner Risk

The booking owner carries temporary cash-flow and refund risk. If one traveler pays for a shared hotel, both travelers should agree on reimbursement timing, cancellation handling, and what happens if one person backs out.

## Output Format

- Individual cost list
- Shared cost list
- Per-person split
- Booking owner
- Cancellation rule
- Reimbursement timing
